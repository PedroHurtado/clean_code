# Clean Code - Estructura del Curso (18 horas / 4 días)

## **DÍA 1 - Fundamentos del Código Limpio (4.5 horas)**

### Sesión Mañana (2.5 horas)
- Introducción al curso y presentaciones (30 min)
- **1. Todo código es mejorable** (2 horas)
  - 1.1. Se escribe para leer
  - 1.2. El paso del tiempo lo empeora todo
  - 1.3. Refactorizar manteniendo la funcionalidad

### Sesión Tarde (2 horas)
- **2. Nombrado** (2 horas)
  - 2.1. Reglas básicas de estilo, tamaños, espacios y comentarios
  - 2.2. Uso de substantivos para representar el modelo
  - 2.3. Uso de verbos para indicar procesos
  - Ejercicios prácticos de renombrado
